# 프로젝트 주제 및 설명 :smile:

요즘 외식을 하게 되는 경우 음식점을 선택하는 트렌드는 검색창에 "내 주변 맛집" 등과 같은 맛집을 키워드로 검색하는 것이다. 하지만 협찬과 광고 등으로 인해 솔직한 음식점 후기를 찾는것이 점점 어려워 짐에 따라 협찬과 광고가 없는 음식점 후기만을 위한 웹 사이트를 개발하는 것이 목적.
